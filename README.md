# IOTA Sample Client
## Compile
```
$ mvn compile
```
## Unit Test
```
$ mvn test
```
## Conect To Node
```
$ mvn exec:java -Dexec.mainClass="com.iota.ConnectToNode"
```
## Get Transaction Inoformation
```
$ mvn exec:java -Dexec.mainClass="com.iota.GetTransaction"
```
## Send Tokens
```
$ mvn exec:java -Dexec.mainClass="com.iota.TransferFrom
```
